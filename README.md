## General info
Application for managing patients, doctors and planning medical visits.

The app is a student project.

## Technologies
Project is created with:
* PHP 7
* Laravel framework
* MySQL database

## Features
* User registration and login
* CRUD for doctors
* Reporting health problems and assigning to a doctor
* Planning new visits
